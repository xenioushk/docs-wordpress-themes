Installation Requirements

We highly recommended using the latest stable version of WordPress to fixed all known bugs and security issues. For this WordPress theme to work perfectly, you must install the WordPress version 5.5 or higher on your server.

Server Requirements:

  • PHP version 7.4.7 or greater.
  • MySQL version 5.0.15 or greater.

Installation Steps

A WordPress theme can be installed and activated using the FTP and the Zip file uploading method.

  • FTP Upload: Using your FTP client, upload the non-zipped theme folder into the /wp-content/themes/ folder on your server.
  • Zip File Upload: Navigate to Appearance >> Themes >> Add New >> Upload. Go to browse, and select the zipped theme folder. Hit Install Now and the theme will be uploaded and installed.

Step 01: Go to WP admin dashboard and click on Appearence >> Themes >> Add New.

Step 02: Click on Upload Theme.

Step 03: Browse the file senior-care.zip which you'll have from the download and click on Install Now .

Step 04: Activate the theme.

Step 05: Click on Begin Installing Plugins link.

Please install and activate the following pre-packaged plugins:

  • One Click Demo Import. (Required)
  • WPBakery Page Builder (formerly Visual Composer). (Required)
  • Redux Framework. (Required)
  • Senior Care Theme WPBakery Page Builder Addons. (Required).
  • Contact Form 7.(Required)
  • Mailchimp For Wp. (Required)
  • The Events Calendar. (Required)
  • Woocommerce. (Optional, If you want to run a shop install and activate it).
  • Widget Importer & Exporter. (Required, but you can deactivate it after import demo data).
  • WordPress Importer. (Required, but you can deactivate it after import demo data).

N.B: All plugins must be installed and activated if you want your site to look exactly like demo.

Step 06: Select the plugins to Install and click Apply.

Step 07: After install the plugin click on Return to Required Plugins Installer.

Step 08: Select the plugin to Activate and click Apply.

Step 09: After active the plugin click on Return to the Dashboard.

We strongly recommended installing the Senior Care theme on a new WordPress site. Additionally, all the plugins must be installed and activated to import demo data accurately.

Import Demo Data

Usually, creating a website from the beginning is a time-consuming and challenging process. The theme comes with the most popular 'One Click Demo Installation' feature, which allows you to get the site ready precisely like the theme demo within a few minutes.

Few Important Notes

  • Make sure that Senior Care Parent Theme has been installed and activated on your site.
  • Please note that your previous website data and settings may be removed or modified after importing demo data. So, we strongly recommend importing Senior Care theme demo data on a new WordPress site.
  • Usually, importing demo data takes 3-5 minutes to complete the entire process. However, it may take a bit longer to import all the images and demo data from our server. Please keep patience and do not close the browser while importing process is running on the site.
  • You can deactivate the 'One Click Demo Import' plugin after completing the importing task.

Please follow the steps -

Step 01: Make sure you have installed and activated the 'One Click Demo Import' plugin.

Step 02: Next, go to Appearance >> Import Demo Data page.

Step 03: You will get all the demo lists in here. Select your required demo and click on the Import button. Usually, importing demo data takes 3-5 minutes to complete the entire process. Please keep patience and do not close the browser while importing process is running on the site.

Step 04: Once the importing process is completed successfully, you will get a success message.

Step 05: Check Result -

If you face any kind of 500 server issue please check the ' troubleshooting article ' .

Theme Translation

"Senior Care" WordPress theme offers cool translation feature. So, beside the English you can easily translate the theme text for other languages as well. Check following steps to translate "Senior Care" WordPress theme.

Step 01: To translate the theme texts, please download and install Poedit software in your computer. Here goes the download link- https://poedit.net/download

Step 02: Next, navigate to /wp-content/themes/senior-care/languages/. You will get a file named 'senior-care.pot'.

Step 03: For example, we want to translate our theme text in to Dutch language. So, copy 'senior-care.pot' file and rename it to 'de_De.pot'

Step 04: Now, open that newly created 'de_De.pot' file using Poedit software. You will get all the theme texts in there and you just need to add appropriate translated text in Translation box.

Step 05: Finally, save the file and you'll get 'de_De.mo' file inside languages folder.

Troubleshooting

"The package could not be installed. The theme is missing the style.css stylesheet.". How to fix?

You may try to upload complete zip file downloaded from themeforest. Unzip you're downloaded file and you'll get 'senior-care.zip' file. Only upload 'senior-care.zip' file. Issue will be solved.

When I attempt to install it, I received the "Are you sure you want to do this?" error message. How to fix?

It's a server execution time related issue. You can contact with your host provider to increase script execution time. There are two ways you can fix that issue.

  • Way 01: Unzip theme folder & files. Then using FTP upload it in wp-content/themes folder.
  • Way 02: You'll find 'plugins' folder inside 'senior-care' theme. Which contain all the required plugins. Just bring it out from 'senior-care' theme folder then zip the theme folder and upload it like regular WP theme.
How to get visual composer updated version?

Just send us a message via support page link. You'll send you updated version download link.
https://themeforest.net/item/sharai-khana-activism-campaign-wordpress-theme/20787548/support/contact

Can not able to install demo data? What should I do now?

It may be server related issue. You can contact with your hosting provider or contact with us via support page. We'll be happy to assist you.

Getting 404 after import data? What should I do?

You need to flush your permalink settings. To do this Go to WP Admin Panel >> Settings >> Permalinks . Then click "Save" button. Now, check you're pages. Hope issue will be solved.

Upgrade Notes

To update theme you can follow the steps-

  • Download Easy Theme and Plugin Upgrades plugin and install it in your site.
  • Choose the theme file (e.g., senior-care.zip) and click Install Now button.

  • Make sure that your theme is active.

Before updating the theme, we recommend you to make backup of your modified files. Also deactivate the current version before uploading the new theme file.
